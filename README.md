# Digital Brain – A Personalized AI Memory and Recognition System

Digital Brain is a personalized AI system designed to recognize users, store memory about them, and interact intelligently using vision, speech, and contextual information.

## 🚀 Features

* **User Recognition** – Identify users using computer vision.
* **Memory System** – Store and retrieve personalized information about users.
* **Voice Interaction** – Communicate using speech input and output.
* **Vision Processing** – Analyze images and detect people.
* **Web UI** – Interactive interface for monitoring and control.

## 🧠 System Modules

* Memory Management
* Vision Processing
* Voice Interaction
* Web Interface

## ⚙️ Setup

### 1. Clone the repository

```
git clone https://github.com/krupakargurije/Digital-Brain.git
cd Digital-Brain
```

### 2. Install Python dependencies

```
pip install -r requirements.txt
```

### 3. Start the UI

```
cd digital-brain-ui
npm install
npm run dev
```

## ▶️ Usage

Run the main AI system:

```
python main.py
```
