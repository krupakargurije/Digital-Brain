# Digital Brain – A Personalized AI Memory and Recognition System

A personalized AI system for memory and recognition.

## Features

- Memory management
- Vision processing
- Voice interaction
- Web UI

## Setup

1. Install Python dependencies: `pip install -r requirements.txt`
2. For the UI: `cd digital-brain-ui && npm install && npm run dev`

## Usage

Run `python main.py` to start the system.
